== Description ==
In its initial state, WordPress does not have an archive page for posts.
WP Post Has Archive allows you to create an archive page with the URL of your choice.

WordPressは初期状態では post のアーカイブページありません。
WP Post Has Archive は好きなURLでアーカイブページを作ることができます。


== Installation ==
1. Upload the entire `post-has-archive` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The control panel of WP Post Has Archive is in 'Settings > Post Has Archive'.

プラグインをインストール後に、設定の中に「Post Has Archive」が表示されます。
